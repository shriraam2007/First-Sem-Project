# First-Sem-Project
Online Python Quiz Game
1. Description
This project is a simple quiz game that runs in the command line using Python. It challenges the player’s general knowledge with 10 multiple choice questions. The game is interactive, giving instant feedback after each answer and showing the final score once all questions are completed.
The program was created as part of a first semester assignment to practice core programming concepts such as handling user input, using lists and dictionaries, working with loops and conditionals, and organizing code into functions.
2. Features
•	Interactive start: The game begins by asking if the player wants to play.
•	10 questions: A fixed set of multiple choice questions covering different topics.
•	Flexible input: Accepts answers as a, b, c, or d (case insensitive).
•	Instant feedback: Each response is marked right away as correct or incorrect.
•	Score tracking: Keeps a running total and displays the final score at the end.
•	Modular design: Functions are used to keep the code clear and easy to maintain.
3. How to Run the Project
Prerequisites
•	Python 3 must be installed on your computer.
Steps
1.	Clone the repository/Download zip file 
2.	Navigate to the project folder
3.	Run the game
4.	Play the quiz
Follow the instructions shown on screen to answer the questions.
